# WBS 3.4 三视图通用验证基线

## 用户人工验收项

1. 只写用户可观察动作。
2. 只写页面和文档可见结果。
3. 不写读代码动作。
4. 不写 `npm` 命令动作。

## 开发者自测项

1. 单元测试
2. 集成测试
3. 类型检查
4. 构建
5. 关键服务接口验证

## 三视图共性观察能力写法

1. `Knowledge`
   - 选择对象
   - 编辑字段
   - 引用跳转
2. `Graph`
   - 创建节点
   - 创建边
   - 拖拽节点
   - 拖动画布
   - 查看坐标
3. `Tracks`
   - 分组切换
   - 多轨显示
   - 事件块换轨
   - 时间锚点显示
4. `Observation`
   - 切片切换
   - 章节文本显示
   - 相关人物展示
