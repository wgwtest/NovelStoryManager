# 双镜对读型 方案说明

## 套系定位

把两条对象链、两个视角切片或两组关系图放到同一台面上对照阅读。

## 结构性标记

- `双镜对读台`
- `左右对照窗`
- `差异摘要带`

## 外部参照

- [Airtable Detail Comparison](https://support.airtable.com/docs/es/airtable-interface-layout-record-detail)
- [Notion Comparison Layouts](https://www.notion.com/help/timelines)

## 设计转译

1. 产品标题统一使用 `知识编辑器`，项目 `青云劫火录` 只做上下文。
2. 左侧对象库保持长期驻留，但中区主工作面的隐喻完全围绕 `双镜对读型` 重构。
3. 右侧检查器只承载当前对象的辅助详情，不替代中区主工作流。
4. `双镜对读型` 的目标不是换色，而是改变用户处理对象、关系、轨道与切片的方式。
