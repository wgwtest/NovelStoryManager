# 剪辑编排型 方案说明

## 套系定位

把故事切片当作剪辑素材来编排，强调多轨时间尺、轨道头和待排事件篮。

## 结构性标记

- `编排时间尺`
- `轨道头矩阵`
- `待排事件篮`

## 设计转译

1. 产品标题统一使用 `知识编辑器`，项目 `青云劫火录` 只做上下文。
2. 左侧对象库保持长期驻留，但中区主工作面的隐喻完全围绕 `剪辑编排型` 重构。
3. 右侧检查器只承载当前对象的辅助详情，不替代中区主工作流。
4. `剪辑编排型` 的目标不是换色，而是改变用户处理对象、关系、轨道与输出切片的方式。
