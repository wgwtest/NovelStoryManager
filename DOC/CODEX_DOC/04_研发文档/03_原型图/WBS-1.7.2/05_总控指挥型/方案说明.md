# 总控指挥型 方案说明

## 套系定位

把三主视图统一收拢到一个总控工作面，重点检查覆盖、联动和输出切片。

## 结构性标记

- `全局覆盖总览`
- `联动监视窗`
- `输出切片指挥板`

## 外部参照

- [Notion Timelines](https://www.notion.com/help/timelines)
- [Airtable Interface Designer](https://support.airtable.com/v1/docs/getting-started-with-airtable-interface-designer)

## 设计转译

1. 产品标题统一使用 `知识编辑器`，项目 `青云劫火录` 只做上下文。
2. 左侧对象库保持长期驻留，但中区主工作面的隐喻完全围绕 `总控指挥型` 重构。
3. 右侧检查器只承载当前对象的辅助详情，不替代中区主工作流。
4. `总控指挥型` 的目标不是换色，而是改变用户处理对象、关系、轨道与输出切片的方式。
