# 剧情弧矩阵型 方案说明

## 套系定位

把人物、事件、线索和剧情弧投到同一矩阵里，强调交叉覆盖与空白区域。

## 结构性标记

- `剧情弧矩阵`
- `角色交叉轴`
- `事件覆盖格`

## 外部参照

- [Miro Matrix / Mapping](https://help.miro.com/hc/en-us/articles/20185235301650-Timeline)
- [Notion Databases](https://www.notion.com/help/timelines)

## 设计转译

1. 产品标题统一使用 `知识编辑器`，项目 `青云劫火录` 只做上下文。
2. 左侧对象库保持长期驻留，但中区主工作面的隐喻完全围绕 `剧情弧矩阵型` 重构。
3. 右侧检查器只承载当前对象的辅助详情，不替代中区主工作流。
4. `剧情弧矩阵型` 的目标不是换色，而是改变用户处理对象、关系、轨道与切片的方式。
