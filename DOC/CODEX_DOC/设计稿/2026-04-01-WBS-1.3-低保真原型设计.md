## 低保真原型需要标出的关键点

1. 左侧对象库始终存在，不因为切换主视图而消失
2. 主视图始终通过 `Knowledge / Graph / Tracks` 三个标签切换
3. 右侧检查器承担对象、边、轨道块的详情编辑
4. Graph 和 Tracks 都需要明确显示拖拽入口和画布区域

## 需要下一节点重点验证的交互热点

1. 从左侧对象库拖拽对象进入 Graph 画布
2. 在 Graph 中拖拽节点、创建关系边、编辑边
3. 从左侧对象库拖拽事件或对象进入 Tracks
4. 在 Tracks 中跨轨移动、排序、吸附和调整块位置
5. 画布缩放、平移、多选、框选是否顺手
