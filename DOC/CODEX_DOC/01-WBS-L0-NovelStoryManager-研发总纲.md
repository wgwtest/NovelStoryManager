# WBS L0 研发总纲

## 项目目标

构建一个面向单本小说项目的文件化剧情建模器，以对象库为真源，以事件与关系为动态骨架，以知识库视图、关系图视图和轨道视图为主要工作面，为后续观察输出和章节切片打底。

## 顶层 WBS

### WBS 1. 需求与方案

1. 产品边界确认与文档归档
2. 技术方案选型
3. 初始任务拆解与时间规划

### WBS 2. 核心对象模型与文件化项目

1. schema v1 定义
2. 项目目录结构定义
3. 文件读写与校验
4. 导入导出协议初版

### WBS 3. 知识库主视图

1. 左侧对象库常驻
2. 列表与筛选
3. 对象详情与字段编辑
4. 对象引用跳转

### WBS 4. 关系图工作台

1. 图节点与关系边渲染
2. 子图浏览
3. 图布局保存
4. 图上关系编辑

### WBS 5. 多轨编排视图

1. 轨道分组策略
2. 事件块编排
3. 预设保存
4. 事实与编排分离

### WBS 6. 观察输出层

1. 时间切片
2. 人物切片
3. 地点切片
4. 剧情线切片
5. 未来章节维度承接

## 推进方式

1. 顶层导航只使用 WBS 分解
2. 任务顺序通过 `Depends On` 表达
3. 时间安排通过 `Start Date` 与 `Target Date` 表达
4. 详细实施计划必须挂靠到 WBS 节点
