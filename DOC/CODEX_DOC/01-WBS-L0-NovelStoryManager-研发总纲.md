# WBS L0 研发总纲

## 项目目标

构建一个面向单本小说项目的文件化剧情建模器，以对象库为真源，以事件与关系为动态骨架，以知识库视图、关系图视图和轨道视图为主要工作面，为后续观察输出和章节切片打底。

## 顶层 WBS

### WBS 1. 项目启动与工程基线

1. 文档根与协作规则建立
2. 仓库运行骨架建立
3. M1 计划前置整理

### WBS 2. 核心对象模型与文件化项目

1. schema v1 定义
2. 项目目录结构定义
3. 文件读写与校验
4. 导入导出协议初版

### WBS 3. 知识库主视图

1. 左侧对象库常驻
2. 列表与筛选
3. 对象详情与字段编辑
4. 对象引用跳转

### WBS 4. 关系图工作台

1. 图节点与关系边渲染
2. 子图浏览
3. 图布局保存
4. 图上关系编辑

### WBS 5. 多轨编排视图

1. 轨道分组策略
2. 事件块编排
3. 预设保存
4. 事实与编排分离

### WBS 6. 观察输出层

1. 时间切片
2. 人物切片
3. 地点切片
4. 剧情线切片
5. 未来章节维度承接

## 里程碑验证门

1. `M1`：项目底座可运行，对象库与文件化项目可用
2. `M2`：关系图工作台可用
3. `M3`：多轨编排视图可用
4. `M4`：观察输出层可用

说明：

1. 里程碑是验证门，不是顶层 WBS 命名轴
2. 详细实施计划必须挂靠到 WBS 节点
