## 协作规则

1. 先读最新稳定计划和设计稿，再进入实现。
2. 稳定计划按 WBS 节点命名，不使用“当前阶段计划”作为稳定文件名。
3. 里程碑 `M1 / M2 / M3 / M4` 仅作为验证门，不作为顶层分解轴。
4. 用户未明确验收前，阶段状态只记为 `待人工验收` 或 `待用户确认`。
5. 验收、自测、交接文档分别进入固定目录，不散落在文档根。
