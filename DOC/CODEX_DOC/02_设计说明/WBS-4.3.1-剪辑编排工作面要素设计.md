# WBS 4.3.1 剪辑编排工作面要素设计

## 设计前提

本文件只回答“剪辑编排页面由什么构成”，不负责解释剪辑编排工作面的产品定位。父级定位见 `WBS-4.3-剪辑编排工作面设计.md`。

## 页面构成

1. 编排工具栏
2. 时间尺
3. 轨道头
4. 轨道体
5. 片段块
6. 片段篮与观察输出联动区

## 数据映射

1. 片段块引用 `objects/*.json` 中的对象或事件。
2. 轨道顺序、片段摆放、切片结果与当前视口属于视图状态。
3. 章节切片来自编排观察结果，不反向定义对象事实。

## 必备能力

1. 时间尺与轨道显示
2. 分组切换
3. 片段块入轨
4. 片段块换轨
5. 轨道顺序调整
6. 视口平移与缩放
7. 事件时间锚点展示
8. 片段篮与轨道之间的拖拽反馈

## 设计约束

1. 页面心智必须接近剪辑软件，不允许退化成普通表格。
2. 时间线、空间线、观察线都应能在同一套轨道体系中表达。
3. 任何拖拽反馈都必须明确区分“预放置”“已入轨”“非法落点”三种状态。

## 要素清单

1. 时间尺：统一定义时间基准和缩放感知
2. 轨道头：表达轨道类型、名称、锁定或分组状态
3. 轨道体：承载片段块与空位
4. 片段块：表达对象、事件或切片的投影单元
5. 片段篮：承载待入轨素材与模板
6. 章节切片区：承载从编排中导出的观察结果
