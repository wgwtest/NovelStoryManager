# WBS 7.1 观察切片模型设计

## 观察模式

1. 时间切片
2. 人物切片
3. 地点切片
4. 剧情切片
5. 章节切片

## 章节切片定义

1. 章节不是对象真源。
2. 章节是观察层存储的一个切片对象。
3. 章节切片保存：
   - `title`
   - `summary`
   - `text`
   - `eventRefs`
   - `focusObjectRefs`
   - `sourceMode`

## 输出要求

1. 每个章节切片都能显示：
   - 覆盖事件
   - 相关人物
   - 相关地点
   - 相关剧情弧
2. 章节文本允许直接在观察层记录和保存。
