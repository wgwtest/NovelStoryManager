# WBS 3.3 通用编辑操作与存储协议

## 编辑分类

1. 改对象真源：
   - 创建对象
   - 修改对象字段
   - 创建关系
   - 修改事件关联
2. 改视图状态：
   - 图节点位置
   - 图画布尺寸与视口
   - 轨道顺序与视口
   - 章节观察切片

## 服务接口约束

1. 对象写入接口：
   - 创建对象
   - 更新对象
   - 创建关系
   - 更新事件轨道关联
2. 视图写入接口：
   - 更新图布局
   - 更新轨道预设
   - 更新章节切片

## 文件落点

1. `views/graph-layouts.json`
   - 图节点位置
   - 图视口
   - 图画布尺寸
2. `views/track-presets.json`
   - 轨道分组
   - 轨道顺序
   - 轨道视口
3. `views/chapter-slices.json`
   - 章节切片
   - 章节文本
   - 章节关联事件与焦点对象
