# WBS 5.1 关系图要素设计

## 页面构成

1. 图工具栏
2. 图画布
3. 边图层
4. 节点图层
5. 坐标与联通状态提示
6. 节点属性面板

## 节点风格基线

1. 节点整体风格参考 Unreal 蓝图节点，而不是普通矩形卡片。
2. 节点至少分为：
   - 标题条
   - 分类色带或类型标记
   - 左输入端口列
   - 右输出端口列
   - 中部参数区
   - 节点底部状态区
3. 节点端口必须可见，不允许把连接点隐藏成纯悬浮热区。
4. 节点选中、悬停、脏状态和非法连接目标都要有明确视觉反馈。

## 端口与连线

1. 左侧端口默认承担输入。
2. 右侧端口默认承担输出。
3. 端口要区分执行流端口和数据端口。
4. 端口周围应存在可感知的吸附区域，拖线接近时高亮。
5. 连线优先使用平滑曲线，并在悬停和选中时改变描边状态。
6. 非法连接必须给出明确失败反馈，而不是静默不响应。

## 必备能力

1. 创建节点
2. 绘制连线
3. 节点拖拽
4. 画布拖拽
5. 缩放
6. 画布大小配置
7. 坐标显示
8. 节点属性编辑入口

## 工具栏与属性区

1. 工具栏至少包含：
   - 创建节点
   - 缩放控制
   - 重置视口
   - 画布尺寸输入
2. 属性区至少包含：
   - 当前节点基础信息
   - 节点类型
   - 端口摘要
   - 关联对象或字段入口
