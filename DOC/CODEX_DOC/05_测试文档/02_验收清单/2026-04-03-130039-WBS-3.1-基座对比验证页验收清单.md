## 人工验收项

- [ ] 启动前端后，在顶部能看到 `WBS 3.1 Base Lab` 按钮
- [ ] 点击 `WBS 3.1 Base Lab` 后，页面切换到独立验证页，不再占用 `Knowledge / Graph / Tracks` 三个正式 tab
- [ ] 验证页中能同时看到 `DOM + CSS`、`DOM + SVG`、`Canvas` 三块对比卡片
- [ ] 三块对比卡片中都能看到相同的一组验证节点、边、缩放按钮和画布尺寸输入框
- [ ] 在任意一块对比卡片中拖动节点或调整缩放后，能看到坐标或视口信息随之变化
- [ ] 点击 `Back to Workbench` 后，能回到正式工作台，并再次看到 `Knowledge / Graph / Tracks` 三个 tab
- [ ] 设计说明文档已明确写出：验证页是独立观察面、三种比较方案、当前暂定推荐和纯 `Canvas` 的限制
- [ ] 研制计划文档已明确写出：代码成果物路径、独立验证页要求、人工验证动作和命令自测方法
