# WBS 1.8.2 端口高亮与命中提示验收清单

## 验收范围

1. `BaseLab` 独立应用中的 `WBS 3.1 基座对比验证`
2. `DOC/CODEX_DOC/02_设计说明/WBS-3.1-交互画布与渲染基座选型.md`
3. `DOC/CODEX_DOC/03_研制计划/08D-WBS-1.8.2-WBS-3.1实验迁移与入口解耦-研制计划.md`
4. `DOC/CODEX_DOC/03_研制计划/13-WBS-3.1-交互画布与渲染基座选型-研制计划.md`

## 人工验收项

- [ ] 在 `http://127.0.0.1:4174/` 打开 `BaseLab` 后，仍可进入 `WBS 3.1 基座对比验证`
- [ ] 在 `DOM + CSS` 画布中，把鼠标移动到任一端口上时，端口应出现明显悬停提示
- [ ] 在 `DOM + SVG` 画布中，从一个输出端口起拖连线时，源端口应持续高亮，移到目标输入端口上时目标端口应出现明显命中提示
- [ ] 在 `Canvas` 画布中，把鼠标移动到端口上时，应能看到明显的光环或描边提示
- [ ] 在 `Canvas` 画布中，从一个输出端口起拖连线并移到目标输入端口上时，应能区分源端高亮和目标端提示
- [ ] 三块验证画布中的拖拽稳定性和动态连线能力未因本轮高亮改动退化
- [ ] 设计说明中已明确写出三种基座在端口提示上的可行性、实现开销和稳定性判断
- [ ] `08D-WBS-1.8.2-WBS-3.1实验迁移与入口解耦-研制计划.md` 与 `13-WBS-3.1-交互画布与渲染基座选型-研制计划.md` 已明确写出统一端口状态模型和人工验收方法
