# WBS 3-7 三视图共性至观察输出层验收清单

## 验收范围

1. `Knowledge` 页
2. `Graph` 页
3. `Tracks` 页
4. `Observation Output` 区

## 人工验收项

- [ ] 在样例工程加载完成后，左侧 `Object Library` 持续驻留，切换 `Knowledge / Graph / Tracks` 时不会消失
- [ ] 在 `Knowledge` 页点击对象类型、输入筛选关键字后，对象列表和中部内容会同步切换
- [ ] 在 `Knowledge` 页点击 `Create Object` 后，左侧对象库出现新对象，右侧检查器切到该对象
- [ ] 在右侧检查器修改对象字段并保存后，可直接看到字段值更新
- [ ] 在 `Knowledge` 页点击引用按钮后，能跳到对应对象类型并选中目标对象
- [ ] 切到 `Graph` 页后，可看到图工具栏、图画布、边和节点
- [ ] 在 `Graph` 页点击 `Create Node` 后，图上出现新节点，并可在右侧检查器编辑
- [ ] 在 `Graph` 页拖拽节点后，顶部坐标显示会变化；点击 `Save Layout` 后状态提示显示保存成功
- [ ] 在 `Graph` 页开启 `Draw Edge` 后，点击另一个节点可新增一条关系边
- [ ] 在 `Tracks` 页可看到轨道工具栏、时间锚点、轨道头和事件块
- [ ] 在 `Tracks` 页切换 `Track Grouping` 为 `character / location / arc` 时，轨道分组会变化
- [ ] 在 `Tracks` 页点击 `Create Event` 后，右侧检查器切到新事件对象
- [ ] 在 `Tracks` 页拖拽事件块到另一条轨道后，事件块位置会变化，右侧检查器中的对应引用字段也会更新
- [ ] 在 `Tracks` 页调整轨道顺序并点击 `Save Preset` 后，状态提示显示保存成功
- [ ] `Observation Output` 区可切换 `Time / Character / Location / Arc / Chapter` 五种观察模式
- [ ] 切换到 `Chapter` 模式后，可看到章节切片卡片，且卡片中能看到覆盖事件、相关人物、相关地点和相关剧情弧
- [ ] 在 `Chapter` 模式修改 `Chapter Summary` 或 `Chapter Text` 并保存后，状态提示显示保存成功
- [ ] 章节观察的编辑不会改变左侧对象库中的对象总数，也不会把章节变成对象类型
