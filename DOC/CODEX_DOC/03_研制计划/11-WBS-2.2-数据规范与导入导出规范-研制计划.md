## 验收方法

- 人工核验规范正文已存在，并至少包含“内部项目文件规范”“项目级导入导出 bundle”“对象级导入导出 bundle”“版本与失败规则”四节。
- 人工核验 `fixtures/import-export/` 下存在本节点约定的正样例和反样例文件。
- 运行 `npm run test --workspace @novelstory/schema`，预期项目级 bundle、对象级 bundle、缺字段、坏引用、版本不兼容、重复对象等测试全部通过，退出码为 `0`。
- 运行 `npm run test --workspace @novelstory/service`，预期文件项目与 bundle 之间的导入、导出、往返一致性测试全部通过，退出码为 `0`。
- 运行 `npm run typecheck`、`npm run build`、`npm run test`，预期根工作区退出码均为 `0`。
