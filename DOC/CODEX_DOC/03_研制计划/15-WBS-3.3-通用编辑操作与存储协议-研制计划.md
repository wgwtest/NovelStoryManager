## 工作目标

- 定义三视图在创建、编辑、删除、拖拽、保存、撤销预留和导入导出时的统一操作语义与存储协议。

## 工作内容

- 明确哪些编辑直接改对象库，哪些编辑只改视图布局或预设。
- 明确视图保存文件、命名规则和版本边界。
- 明确服务层接口的最小读写动作。

## 实施方式

- 以“对象真源不被视图污染”为第一约束。
- 对无法立即实现撤销等高级能力的部分，先定义预留接口和不支持边界。
