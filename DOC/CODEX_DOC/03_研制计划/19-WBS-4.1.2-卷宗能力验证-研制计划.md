## 工作目标

- 先在 `BaseLab` 的独立卷宗页中证明卷宗工作面具备真正可观察的基础能力，并且能向审阅者解释卷宗为什么这样设计、需要什么样的数据，而不是只展示一个静态页面或一张内部操作台。

## 工作内容

- 在 `BaseLab` 中实现独立卷宗验证页与模拟对象数据装载。
- 在独立页中补充 `设计说明 / 样例规范 / 工作面验证` 三层入口。
- 修正并验证对象筛选、对象自动切换和类型切换。
- 修正并验证引用点击后能切到目标类型并显示目标对象。
- 修正并验证回查按钮文案清晰可见，且能从目标对象返回来源对象。
- 修正并验证检查器分组可见、可滚动、可操作。
- 补齐 `Reference Fields` 分组，使 `factionRefs / sourceRef / targetRef` 等引用字段可直接编辑。
- 补齐 `关系档案 / 出场记录 / 关系草拟` 三类能力，让卷宗不再停留在只读查阅。

## 实施方式

- 当前 `苏玄 -> 青云宗` 引用跳转问题和卷宗回查区布局问题，在本节点优先解决。
- 本节点只验证卷宗工作面自身能力，不承担主工作台集成。
- 验证项必须写成用户可观察动作，不使用抽象描述。
