# WBS 6.1 观察输出层

工作目标：

- 基于对象和事件网络生成时间切片、人物切片、地点切片和剧情线切片，并在 Tracks 工作台中提供派生观察面板，同时为未来章节观察维度预留承接点。

Owner：

- Codex

Write Scope：

- `DOC/CODEX_DOC/开发计划/08-WBS-L2-WBS-6.1-观察输出层计划.md`
- `apps/web/src/lib/project-observation.ts`
- `apps/web/src/components/TracksView.tsx`
- `apps/web/src/App.test.tsx`
- `apps/web/src/styles.css`

工作内容：

- 实现四类切片派生接口。
- 在 Tracks 工作台中提供观察结果面板。
- 明确观察输出结果和对象事实之间的单向派生关系。
- 预留未来章节观察维度的接入位置。

潜在难点：

- 切片逻辑一旦反向写回模型，就会破坏对象库真源。
- 不同切片之间共享筛选条件时，容易造成状态复杂度上升。

- `WBS 2.1` 项目底座实施
- `WBS 5.1` 多轨编排视图
