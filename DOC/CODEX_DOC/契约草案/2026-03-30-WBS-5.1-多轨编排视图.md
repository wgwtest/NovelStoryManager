# WBS 5.1 多轨编排视图

目标：

- 构建按地点、剧情线、角色视角、势力等维度切换的多轨事件编排视图。

Owner：

- Codex

Write Scope：

- Tracks tab 前端模块
- 轨道预设接口
- 轨道状态持久化

Depends On：

- `WBS 2.1 M1 项目底座实施`
- `WBS 4.1 关系图工作台` 可并行 Prep，不依赖其正式关闭

验收入口：

- 主入口：Tracks tab
- 辅助入口：轨道预设保存与加载

验收标准：

- 支持多种轨道分组策略
- 支持事件块展示与编排
- 明确区分视图编排和事实修改

输出物：

- Tracks tab 可用版本
- 轨道预设持久化
