# WBS 3.1 知识库主视图增强

工作目标：

- 在项目底座已有的知识库界面上，补齐筛选、字段分组、批量维护和对象引用跳转能力。

工作内容：

- 为对象列表增加类型切换后的筛选能力。
- 为详情页增加字段分组和更明确的编辑布局。
- 为对象之间的关联字段提供基础跳转能力。
- 如有需要，补齐对应 API 和前端测试。

潜在难点：

- 交互增强容易让知识库视图承担过多领域逻辑。
- 引用跳转若没有稳定对象标识，会导致跳转链路不可靠。

Depends On：

- `WBS 2.1` 项目底座实施

成果物：

- `apps/web/src/components/ObjectLibrary.tsx`
- `apps/web/src/components/KnowledgeView.tsx`
- 相关 API 与测试

验收方法：

- 运行 `npm run test --workspace @novelstory/web`，预期退出码为 `0`。
- 启动前端后，人工核验：
  - 可以按对象类型切换并筛选
  - 详情区字段具备分组展示
  - 对象引用字段可以跳转到目标对象
