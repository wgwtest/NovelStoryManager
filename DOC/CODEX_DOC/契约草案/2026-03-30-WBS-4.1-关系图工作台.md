# WBS 4.1 关系图工作台

工作目标：

- 实现对象关系图视图首个可用版本，支持节点展示、关系边读取、局部子图查看和布局持久化。

工作内容：

- 渲染对象节点和关系边。
- 支持局部子图浏览或过滤。
- 支持图布局拖拽与保存。
- 将图布局写入独立的 `views/graph-layouts.json`。

潜在难点：

- 图布局状态和关系事实容易被写到同一份数据中。
- 大图渲染和子图切换可能带来交互复杂度和性能问题。
