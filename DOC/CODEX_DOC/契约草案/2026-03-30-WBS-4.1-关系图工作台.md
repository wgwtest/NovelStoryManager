# WBS 4.1 关系图工作台

工作目标：

- 实现对象关系图视图，支持节点展示、关系边编辑、局部子图查看和布局持久化。

工作内容：

- 渲染对象节点和关系边。
- 支持局部子图浏览或过滤。
- 支持关系边创建、修改或删除。
- 将图布局写入独立的 `views/graph-layouts.json`。

潜在难点：

- 图布局状态和关系事实容易被写到同一份数据中。
- 大图渲染和子图切换可能带来交互复杂度和性能问题。

Depends On：

- `WBS 2.1` 项目底座实施

成果物：

- Graph 视图前端模块
- 关系读写接口
- `views/graph-layouts.json` 持久化链路

验收方法：

- 运行 `npm run test --workspace @novelstory/web`，预期退出码为 `0`。
- 如服务层有关系接口测试，运行对应测试命令，预期退出码为 `0`。
- 启动应用后，人工核验：
  - Graph tab 可渲染节点和关系边
  - 可以查看局部子图
  - 布局修改后可重新加载
