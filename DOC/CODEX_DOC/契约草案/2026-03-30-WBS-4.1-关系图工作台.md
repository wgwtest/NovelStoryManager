# WBS 4.1 关系图工作台

目标：

- 构建对象关系可视化工作台，支持节点布局、关系边编辑和局部子图查看。

Owner：

- Codex

Write Scope：

- 关系图前端模块
- 关系读写接口
- 图布局状态存储

Depends On：

- `WBS 2.1 M1 项目底座实施`

验收入口：

- 主入口：Graph tab
- 辅助入口：关系边创建与布局保存

验收标准：

- 支持节点与关系边渲染
- 支持局部子图浏览
- 图布局单独写入 `views/graph-layouts.json`

输出物：

- Graph tab 可用版本
- 布局持久化链路
